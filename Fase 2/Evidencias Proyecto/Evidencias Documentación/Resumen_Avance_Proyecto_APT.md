# RESUMEN DE AVANCE PROYECTO APT
## Sistema de Gestión de Procesos de Reclutamiento y Selección
### CAPSTONE_001D - Grupo 6

---

Durante el desarrollo del proyecto APT se han logrado avances significativos en las etapas de documentación, diseño, desarrollo e implementación de la aplicación web para la gestión integral de procesos de reclutamiento, selección y evaluación psicolaboral de LL Consulting.

En la fase inicial se elaboraron los principales documentos de planificación: Acta de constitución, Documento de Alcance, Especificación de Requerimientos de Software (ERS), modelo de base de datos y prototipos de interfaz, los cuales establecieron las bases conceptuales y funcionales del sistema. Adicionalmente, se documentaron propuestas para un módulo analítico con 22 categorías de análisis que permitirán la toma de decisiones basada en datos.

Posteriormente, se avanzó en la implementación del backend bajo una arquitectura en capas (controladores, servicios y modelos) utilizando Node.js con Express y TypeScript, utilizando PostgreSQL como base de datos y Sequelize ORM para la gestión de las entidades y relaciones. El sistema incluye autenticación JWT, control de roles (administrador y consultor), validaciones de datos, manejo de archivos y un sistema completo de auditoría mediante triggers de base de datos que registran todos los cambios con el usuario responsable.

El backend implementado abarca 39 modelos de datos que cubren todas las entidades del dominio: usuarios, clientes, contactos, solicitudes, candidatos, postulaciones, hitos, evaluaciones psicolaborales, estados y relaciones entre estas entidades. Se han desarrollado 25 controladores y 22 servicios especializados que gestionan la lógica de negocio, incluyendo cálculos automáticos de plazos máximos por tipo de servicio, generación automática de hitos basados en plantillas, y gestión completa del ciclo de vida de los procesos de reclutamiento.

En paralelo, se desarrolló el frontend utilizando Next.js, React y TailwindCSS, implementando los perfiles de administrador y consultor con funcionalidades completas. El módulo de administración permite la gestión integral de solicitudes, clientes, usuarios y reportes, incluyendo un dashboard con estadísticas en tiempo real (total de solicitudes, en progreso, completadas y pendientes), filtros avanzados por estado, tipo de servicio y consultor, y un sistema de alertas que muestra los hitos próximos a vencer y vencidos, con capacidad de filtrar por consultor y tipo de servicio.

El módulo de consultor incluye un dashboard personalizado, visualización de procesos asignados, y la implementación completa de los cinco módulos del proceso de reclutamiento: Módulo 1 (Publicación de cargos y gestión de portales), Módulo 2 (Recepción y evaluación de candidatos), Módulo 3 (Presentación de candidatos al cliente), Módulo 4 (Evaluación psicolaboral) y Módulo 5 (Gestión de contratación). Cada módulo cuenta con interfaces específicas, validaciones de negocio y seguimiento de estados. Se implementó una timeline visual que muestra los hitos reales de cada proceso, permitiendo a los consultores visualizar el progreso y cumplimiento de las actividades planificadas.

Se completó la integración total entre frontend y backend para todos los módulos, incluyendo la gestión de solicitudes, registro y evaluación de candidatos, presentación al cliente, evaluaciones psicolaborales y seguimiento de contrataciones. Se implementó un sistema de hitos y alertas que genera automáticamente hitos basados en plantillas por tipo de servicio, calcula fechas límite considerando días hábiles (excluyendo fines de semana y feriados), y proporciona alertas proactivas cuando los hitos están próximos a vencer o ya han vencido. El sistema excluye automáticamente alertas de procesos congelados o cancelados para mantener la relevancia de las notificaciones, y permite a los administradores visualizar el nombre del consultor y el tipo de servicio en cada alerta.

Se implementaron mejoras significativas en el sistema de auditoría, corrigiendo registros que mostraban usuarios genéricos ("avnadmin") y asegurando que todas las operaciones críticas (creación, actualización y eliminación) en tablas como `solicitud`, `postulacion`, `candidato`, `estado_solicitud_hist`, `hito_solicitud`, `evaluacion_psicolaboral`, `estado_cliente_postulacion` y `estado_cliente_postulacion_m5` registren correctamente el usuario responsable mediante la utilidad `setDatabaseUser`. Se optimizó el rendimiento del frontend mediante la implementación de un sistema de caché en memoria para datos estáticos (regiones, comunas, profesiones, instituciones, portales, estados de solicitud, etc.) con TTL configurable de 10 horas, reduciendo significativamente las llamadas redundantes a la API.

Se establecieron duraciones fijas por tipo de servicio para el cálculo de plazos máximos, reemplazando el enfoque inicial de sumar duraciones de hitos, lo que proporciona mayor precisión y simplicidad en el mantenimiento. Los tipos de servicio implementados incluyen: Proceso Completo (21 días hábiles), Headhunting (26 días hábiles), Evaluación Psicolaboral (2 días hábiles), Evaluación de Potencial (4 días hábiles), Test Psicolaboral (1 día hábil), Long List (11 días hábiles), Filtro Inteligente (9 días hábiles), Target Recruitment (11 días hábiles) y Publicación en Portales (6 días hábiles). El sistema calcula automáticamente el `plazo_maximo_solicitud` sumando estos días hábiles a la fecha de ingreso de la solicitud.

A la fecha, el proyecto ha alcanzado un backend funcional y escalable con 39 modelos, 25 controladores y 22 servicios, una interfaz moderna e intuitiva con más de 50 componentes reutilizables, y una integración estable entre ambas capas. Se han cumplido los objetivos específicos de centralización de información, trazabilidad completa mediante auditoría, automatización de procesos (cálculo de plazos, generación de hitos, alertas), y gestión integral de los cinco módulos del proceso de reclutamiento. El sistema permite la operación completa de los procesos de reclutamiento y selección de LL Consulting, desde la creación de solicitudes hasta la contratación de candidatos.

Queda pendiente para la siguiente etapa la implementación completa del módulo analítico con reportes y visualizaciones de datos basados en las 22 categorías de análisis documentadas, así como funcionalidades adicionales como notificaciones por email, exportación de datos a Excel/PDF, y pruebas automatizadas. Sin embargo, el sistema actual proporciona una base sólida y funcional que cumple con los requisitos principales establecidos en la ERS.

Estos avances reflejan un progreso sostenido y coherente con los objetivos establecidos, consolidando una base técnica y funcional robusta que permite la operación completa del sistema y establece las bases para futuras expansiones y mejoras, incluyendo el módulo analítico que permitirá la toma de decisiones estratégicas basadas en datos históricos.
